# Benedicto College Library System

A computerized library management system developed for Benedicto College ? Main Campus during the Academic Year 2017-2018. The system leverages barcode scanning technology to streamline the borrowing and returning processes of library materials.

## Overview

This automated system replaces the manual library management process, offering improved efficiency and reduced human effort in managing library resources. The system serves students, teachers, and library personnel by providing a modern approach to library operations.

## Features

- Barcode-based book tracking system
- Automated borrowing and returning processes
- User management for students and faculty
- Reference material cataloging
- Transaction tracking and history
- Resource availability monitoring

## Target Users

- Students
